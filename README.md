# FitLog — 食事・運動・体重トラッカー

食事のPFC・運動・体重を記録して可視化する、**スマホ向けの個人用アプリ**です。
サーバー不要・**データは端末内だけに保存**・完全無料・完全オフラインで動きます。

写真解析はこのアプリには含みません。解析は無料の [Claude.ai](https://claude.ai) 側で行い、
その**結果テキストを貼り付けて取り込む**設計です（→「解析結果の貼り付けフォーマット」）。

---

## できること

- **日次の記録**：日付を選んで（前日・翌日に移動）、その日の体重・食事・運動を記録/編集
  - 体重：1日1つ。目標体重との差を表示
  - 食事：複数件。カロリーは `P×4 + F×9 + C×4` で自動計算
  - 運動：複数件。種目名 / 種別（筋トレ・有酸素・その他）/ メモ
  - その日のPFC・カロリー合計を、目標に対する進捗バーで表示
- **グラフ（直近30日）**：体重の折れ線・摂取カロリーの棒・タンパク質の棒（いずれも目標の基準線つき）
- **目標設定**：カロリー / P / F / C / 目標体重 を変更・保存
- **解析結果の貼り付け取り込み**：解析テキストを貼って一括で食事リストに展開
- **バックアップ**：全データを JSON で書き出し／読み込み（機種変更・紛失に備えて）

---

## データはどこにある？

記録は**使っている端末のブラウザ内（IndexedDB）だけ**に保存されます。
サーバーにもクラウドにも送りません。だから無料で、プライバシーも守られます。

ただし「その端末の中だけ」なので、次の点に注意してください：

- 別の端末（PCとスマホ等）とは**データが共有されません**（それぞれ別の記録になります）。
- ブラウザのデータを消す／機種変更すると消えます → **ときどき「書き出し」でバックアップ**を。

## スマホ（iPhone）で使う

電波のない出先でも使えるよう、ホーム画面に追加してアプリのように使います。
そのためにはアプリを **HTTPS** で一度読み込む必要があります（ブラウザの仕様。
家のWi-FiのLAN IP の素のHTTPだとオフライン化の仕組みが働きません）。
方法は2つあります。

### 方法A：無料の静的ホスティングに置く（手軽・おすすめ）

GitHub Pages / Cloudflare Pages / Netlify などの**無料の静的ホスティング**に、
このフォルダの中身（HTML/JS/CSS/アイコン）をアップロードします。発行されたHTTPSのURLを
iPhone の Safari で開き、共有メニューから「**ホーム画面に追加**」。

- アップロードするのは**アプリのコードだけ**です。あなたの記録データは端末内のみで、外に出ません。
- 一度ホーム画面に追加すれば、以降は**電波が無くても起動・入力**できます。

### 方法B：家庭内で完結させる（コードも外に出したくない場合）

Mac で自己署名のHTTPSサーバーを立て、同じWi-Fiの iPhone から開いて「ホーム画面に追加」します。
iPhone では証明書プロファイルの導入が必要で、方法Aより手間がかかります。手順が必要なら相談してください。

### iPhone での追加手順（共通）

1. Safari でアプリのURLを開く
2. 画面下の「共有」ボタン → 「**ホーム画面に追加**」
3. ホーム画面の FitLog アイコンから起動（全画面のアプリ風に動きます）

## PC で動作確認・お試し

PC のブラウザでも動きます。フォルダ内でローカルサーバーを立てて開いてください
（`127.0.0.1` 経由なら Service Worker も含めて確認できます）。

```bash
cd ~/fitlog
python3 -m http.server 5500 --bind 127.0.0.1
```

ブラウザで **http://127.0.0.1:5500** を開きます。
※ `file://` で直接開くと一部機能（Service Worker）が動きません。上記のように簡易サーバー経由で開いてください。

## 解析結果の貼り付けフォーマット

「食事」カードの「解析結果を貼り付け」欄に、**1行＝1料理**で貼り付けて「取り込む」を押します。

```
メニュー名 | P○ F○ C○
```

- 左が**メニュー名**、`|` の右が**栄養素**。
- P・F・Cは**順不同**でOK、**空白区切り**。数値は小数も可。
- 書かなかった栄養素は 0 として扱います。

### 例

```
鶏むね焼き | P40 F4 C0
白米 茶碗1杯 | P4 F0 C55
味噌汁 | P3 F2 C6
```

→ 3件が食事リストに展開され、カロリーは自動計算されます。
取り込んだあとも、各料理のP・F・C・メニュー名は手で修正・削除できます。

### 読めなかった行について

形式が崩れた行（`|` が無い・メニュー名が空・P/F/Cが一つも無い等）は**取り込まず**、
「○行目: （その行の内容）」と一覧で知らせます。直して貼り直してください。
正しく読めた行だけは先に取り込まれます。

> 数値を直接入力する従来の入力欄（メニュー名 / P / F / C → 追加）も併用できます。

## バックアップと復元

- **書き出し**：目標タブ →「データのバックアップ」→「書き出し（JSON）」。
  `fitlog-backup-YYYY-MM-DD.json` がダウンロードされます。安全な場所に保管してください。
- **読み込み**：同じ場所の「読み込み」からバックアップJSONを選択。
  **現在のデータは置き換わります**（確認ダイアログあり）。
- 機種変更時：旧端末で書き出し → 新端末のFitLogで読み込み、でデータを移せます。
